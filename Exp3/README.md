# ChoiceAdaptationModels_NHB

## Repository Structure
Folder 'Exp3' contains file 'data_loss.mat' containing data of all participants in Experiment 3 and a subfolder 'losspar' containing scripts to fit models to choices of any one participant.


## Instructions to use
Navigate to subfolder 'losspar' to run model fitting to data of any one participant in Experiment 3.
Follow instructions in file 'README.md' in subfolder 'losspar'.
